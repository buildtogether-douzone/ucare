## U-Care Project

### HIS(Healthcare Information System)

#### 다운

```bash
$ git clone https://github.com/buildtogether-douzone/ucare.git
$ cd ucare
$ npm i
```

#### 개발환경

+ Front-End: React
+ Back-End: Spring Boot
+ Maven
+ Java 8
+ PostgreSQL
